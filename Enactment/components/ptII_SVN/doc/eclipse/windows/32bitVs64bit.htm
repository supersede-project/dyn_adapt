32-bit or 64-bit

The basic idea is that to use a 64-bit version of Eclipse, you must have a 64-bit JVM and an operating system that will support 64-bit applications.

When in doubt, choose the 64-bit version. However, one reason to chooose a 32-bit version would be if you are interfacing to a 32-bit version of Dymola using FMI.

32-bit or 64-bit?

  • Windows 8: Oddly, Microsoft does not appear to have instructions for how to determine if Windows 8 is 32-bit. Instead, see http://pcsupport.about.com/od/windows-8/a/windows-8-64-bit-32-bit.htm for details
  • Windows 7 or Vista: "Start", then right click on "Computer", then click on "Properties". Under System, you should see the system type. The 64-bit version of Windows 7 will say "64-bit Operating System". See http://windows.microsoft.com/en-US/windows7/32-bit-and-64-bit-Windows-frequently-asked-questions. for details.
