This directory contains tools for use with Ptolemy Classic.

harvest - Scripts that harvest preexisting Ptolemy Classic .pl files
	and generate .java files

ptIIlang - Yacc tool based on Ptolemy Classic ptlang that reads a
           single .pl file and generates a .java file.
