This directory contains scripts that process Ptolemy Classic .pl files
and generates .java files.

fixclassic - a script that does a half hearted attempt at converting
  Ptolemy Classic .c code to .java code.  fixclassic is called by
  the harvest script.

harvest - a script that processes multiple .pl files.

Most users will edit and run harvest.
